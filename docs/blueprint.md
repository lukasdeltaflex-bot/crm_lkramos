# **App Name**: INSS Manager

## Core Features:

- Dynamic Customer Registration: Register customer data (name, CPF, phone, email) once and use it across all modules.
- Loan Proposal Management: Register loan proposals including table, term, installment amount, net amount (released), gross amount (contract value).
- Commission Tracking: Record commission received, payment date, and track pending payments.
- Averaged, in progress, rejected report: Visualize reports with data concerning proposals.
- Financial Dashboard: View summary reports of receivables, commissions earned and paid.
- Customer Anniversary Alert: Set up special alerts for customers turning 75 years old to comply with lending policies, used an ai tool for dynamic alerts. 
- Product categorization: Categorize products like Margin, CLT Margin, Supplementary Withdrawal, Plastic Card, Portability, Refinancing, FGTS Withdrawal, etc.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust and professionalism, as commonly used in the finance sector.
- Background color: Light blue-gray (#F0F4F8), a desaturated variation of the primary, for a clean and professional look.
- Accent color: Soft teal (#38B2AC), an analogous color to the primary, used sparingly to highlight important actions and information.
- Body text: 'PT Sans', a humanist sans-serif suitable for longer lines of text.
- Headline text: 'Playfair', a modern serif font that pairs well with the sans-serif body.
- Use clean, professional icons to represent different loan products, financial data, and customer information. Ensure icons are easily recognizable and intuitive.
- Employ a clean and structured layout with clear visual hierarchy to allow quick access to information
- Incorporate subtle transitions and loading animations to improve the user experience, such as a loading animation on charts.
